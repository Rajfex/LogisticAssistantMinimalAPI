# Logistic Assistant
LogisticsAssistant is a web application built with ASP.NET Core Minimal API, designed for fleet management and route scheduling. It allows users to manage vehicles, plan delivery routes.

## Features
- **User Authentication**: Secure registration and login system for managing private fleet data.
- **Fleet Management**: Full CRUD operations for adding and managing transport vehicles.
- **Route Planning**: Create and track routes with automated duration calculations.

## Technologiesa
- **Backend**: ASP.NET Core Minimal API
- **Database**: SQL Server

## Installation and Setup
Clone the repository:
``` bash
git clone https://github.com/Rajfex/LogisticAssistantMinimalAPI.git
```
Open the solution file ```(.sln)``` in Visual Studio.

Build and run the application.
